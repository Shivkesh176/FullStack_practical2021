CSS

Cascading Style Sheets, fondly known as CSS, is a simply designed language intended to simplify the process of making web pages presentable.CSS allows you to apply styles to web pages.More importantly, CSS enables you to do this independent of the HTML that makes up each web page.CSS is designed to enable the separation of presentation and content, including layout,colors and fonts.
There are three types of CSS:

  • Inline CSS
  • Internal or Embedded CSS
  • External CSS
