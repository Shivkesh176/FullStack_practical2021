Visual Studio Code

Visual Studio Code(famously known as VS Code) is a free open source text editor by Microsoft.VS Code is available for Windows,Linux and macOS.Although the editor is relatively lightweight,it includes some powerful feature that have made VS Code one of the most popular development environment tools in recent times.

Features

VS Code supports a wide array of programming languages from Java, C++ and Pythin to CSS,Go,and Dockerfile. Moreover VS Code allows you to add on and even creating new extensions including code linters,debuggers and cloud and web development support.

To simplify user experience, VS Code is divided into five main regions:

  • The activity bar
  • The side bar
  • Editors groups
  • The panel
  • The status bar
